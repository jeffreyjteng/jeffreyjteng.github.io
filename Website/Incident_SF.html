San Francisco Incident Analysis Project

Personal Project                                                                                                                                                                                                   July 2021 - August 2021
Excel, MySQL, Tableau

Project Information

SF Incident Data Analysis project focus on few questions such as: What are the incidents that occurred the most?, what are the neighborhoods with the most incidents occurred, does time of the year affect the incidents, are incidents emergency or non-emergency, and when and where will be better to increase the number of police officer to improve the safety.

Software

Software used in this project are Excel, MySQL, and Tableau
Excel was used to clean the data and transform the data into consistment format in order to better import into MySQL databases
MySQL was used to import, clean, process, extract, and export data. MySQL was used to analyzed the data with different conditions
Tableau was used for data visualization, a dashboard with multiple visualizations
