Divvy Bike Share Data Analytic Project

Google Capstone Project                                                                                                                                                                                July 2021 - August 2021
Excel, MySQL, Tableau

Project Information

Divvy is a bike share company located in Chicago, they have three types of plans that the customer can use: single ride, day pass, and annual memberships. For this project, my goal is to analyze the data from the past two years to figure out where, when, and how to advertise the membership program in order to convert more non member user to subscribe the the member plan.

Software

Software used in this project are Excel, MySQL, and Tableau
Excel was used to clean the data and transform the data into consistment format in order to better import into MySQL databases
MySQL was used to import, clean, process, extract, and export data. MySQL was used to analyzed the data with different conditions
Tableau was used for data visualization, a dashboard with multiple visualizations
